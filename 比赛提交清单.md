# 比赛提交清单

| 赛题要求 | 对应交付物 |
|---|---|
| 可运行工具/脚本/Agent | `start.ps1`、`start.bat`、本地 Web、CLI |
| 使用说明 | `使用说明.md`、`app/README.md` |
| 数据证明提效 | `metrics.json`、`compare.html`、样例输出 |
| 低门槛复用 | 上传 `.pptx` 即可得到输出，错误提示中文直说 |
| 创意组合 | LibreOffice 原生转换 + OOXML 动画/遮挡分析 + PyMuPDF PDF 微调重排 + 对比展示 |
| 能力边界 | `report.json` 和文档明确不支持范围 |

## 演示顺序
1. 启动 `start.ps1`
2. 上传 `app/samples/test.pptx`
3. 打开 `compare.html`
4. 展示 `guide.pdf` 的遮挡展开和流程关系
5. 展示 `metrics.json`
6. 说明边界和可扩展 AI 解释接口
