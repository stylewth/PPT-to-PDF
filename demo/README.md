# 课件转讲义助手 Demo

## 定位
面向大学课程课件和企业培训 PPT，将依赖现场讲解的演示型 PPT 转成可阅读、可批注、可复习的学习型 PDF。

## 打开方式
直接用浏览器打开 `index.html`。

## 当前能力
| 能力 | 状态 |
|---|---|
| 上传 PPT 入口 | 已完成交互入口 |
| 大学课程/企业培训场景切换 | 已完成 |
| 学习版/批注版/回看版输出切换 | 已完成 |
| 动画路径、遮挡、解释缺失的结果展示 | 已完成演示版 |
| 浏览器打印导出 PDF | 已完成 |

## 当前边界
这是前端交互 Demo，展示产品流程和输出形态；当前版本不解析真实 PPT 动画 XML，不生成真实 PDF 二进制文件。正式版本需要后端解析 PPTX 结构、动画顺序、对象层级和字体信息。
