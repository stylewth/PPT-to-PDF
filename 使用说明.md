# Slide2Study 使用说明

## 运行前准备
| 依赖 | 说明 |
|---|---|
| Python 3.10+ | 运行本地服务和解析流程 |
| LibreOffice | 生成原生 `base.pdf` |
| PyMuPDF | 在 `base.pdf` 上做 PDF 微调重排 |

安装 Python 依赖：
```powershell
python -m pip install -r .\app\requirements.txt
```

## 启动
```powershell
powershell -ExecutionPolicy Bypass -File .\start.ps1
```

打开：
```text
http://127.0.0.1:8765
```

## 使用
| 步骤 | 操作 |
|---|---|
| 1 | 上传 `.pptx` 文件 |
| 2 | 等待生成 `base.pdf`、`guide.pdf`、`compare.html` |
| 3 | 先打开 `compare.html` 看普通 PDF 和学习版 PDF 的差异 |
| 4 | 下载 `guide.pdf` 用于复习、批注或培训资料沉淀 |
| 5 | 打开 `report.json` 和 `metrics.json` 查看问题与提效数据 |

## 输出说明
| 文件 | 说明 |
|---|---|
| `base.pdf` | 原生转换结果，用作对照 |
| `guide.pdf` | 学习版 PDF，保留原画面并展开遮挡/流程关系 |
| `compare.html` | 比赛展示页 |
| `analysis.json` | 结构化分析结果 |
| `augment_plan.json` | 重排计划 |
| `metrics.json` | 估算提效指标 |
| `report.json` | 问题报告和能力边界 |

## 能力边界
首版只支持 `.pptx`。不支持 `.ppt`、Keynote、加密文件、宏、音视频还原、复杂触发器和交互按钮。无法可靠处理的页面会保留原生页并写入报告。
