# 3 分钟路演脚本

| 时间 | 内容 |
|---|---|
| 0:00-0:30 | 痛点：普通 PPT 转 PDF 只保留最终画面，动画顺序、遮挡关系和讲师讲解过程会丢失。学生复习和员工培训回看时，需要重新猜流程。 |
| 0:30-1:10 | 演示上传 `test.pptx`。系统先用 LibreOffice 生成 `base.pdf`，再解析 PPTX 中的对象、动画、层级和遮挡风险。 |
| 1:10-1:50 | 打开 `compare.html`。左侧是普通 PDF，右侧是学习版 PDF。重点展示：被遮挡内容被放到清晰位置，优先利用原页空白；同时用编号、箭头或关系线表达流程。 |
| 1:50-2:25 | 打开 `metrics.json` 和 `report.json`。说明工具记录了动画页、遮挡问题、重排页和估算节省时间，能力边界也直接暴露。 |
| 2:25-3:00 | 总结：这是一个 PPT 讲解还原 Agent，不是普通格式转换器。它把动态演示里的讲解过程沉淀成可阅读、可复习、可交付的学习型 PDF。 |
